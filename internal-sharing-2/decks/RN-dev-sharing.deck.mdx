```js live=true
/** Fetch image upload url and show form to submit comment with image */
async function fetchBlobstoreUrlAndShowForm() {
  const response = await fetch("/blobstore-upload-url")
  const imageUploadUrl = await response.text()
  const messageForm = document.getElementById("my-form")
  messageForm.action = imageUploadUrl
  messageForm.classList.remove("blob-store")
}
```

```java live=true
/** Servlet responsible for return url to servlet for user to upload. */
@WebServlet("/blobstore-upload-url")
public class BlobstoreUploadUrlServlet extends HttpServlet {
  private final  BlobstoreService blobstoreService = BlobstoreServiceFactory.getBlobstoreService();

  @Override
  public void doGet(HttpServletRequest request, HttpServletResponse response)
  throws IOException {
    String uploadUrl = blobstoreService.createUploadUrl("/comments");

    response.setContentType("text/html");
    response.getWriter().println(uploadUrl);
  }
```

<h1 style={{ textAlign: "center" }}>10x thinking</h1>

### What it is:

- ~~Google brainwashing~~
- Google's culture of innovation
- To put the idea simply: true innovation happens when you try to improve something by 10 times rather than by 10%.
- Pushes you beyond existing models and forces you to totally reimagine how to approach a problem.
- "10x is easier than 10%"

### Where can you find it:

- Google x - the moonshot company
- Projects like Tidal, Everyday Robot etc

### Primers:

- Really about going the extra mile and not settling
- Strive for excellence
